# MAKO | Food Delivery Platform

MAKO is a professional, responsive food delivery platform designed to provide a seamless ordering experience. It focuses on high performance, clean UI/UX, and real-time backend synchronization.

## 🚀 Key Features
* **Dynamic Product Catalog:** Categorized food items with an efficient browsing experience.
* **Smart Filtering:** Built-in price filtering system (Low to High / High to Low).
* **Real-time Shopping Cart:** Persistent cart management allowing users to track their orders.
* **Firebase Integration:** Uses Firestore for backend data storage and order management.
* **Interactive UI:** Features modern popups and animations for a smooth user experience.
* **Responsive Design:** Fully optimized for mobile and desktop screens.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
* **Backend:** Firebase Firestore (Cloud Database).
* **Styling:** CSS Grid & Flexbox, FontAwesome for icons.
* **Design:** Custom dark-themed modern UI.

## 💡 Why MAKO?
MAKO isn't just a UI project; it represents a functional platform architecture. It demonstrates the ability to manage application state (shopping cart), handle real-time database requests (Firebase), and manipulate DOM elements dynamically based on user interaction.

## 📝 How to run
1. Clone the repository: `git clone https://github.com/mmyasomm-crypto/Mako`
2. Open `index.html` in your browser.
